Clean Up After Yourself

02 Mar 2014

When I was growing up my dad always made sure I followed one rule whenever we were working on a project around the house: clean up once you are done. This makes logical sense. When everything has a place and it is in that place, then it is easy for yourself or anyone else to pick up on that project without needing to search for that [redacted] 11/64" socket. How come software developers then seem to have such a hard time with this concept?
